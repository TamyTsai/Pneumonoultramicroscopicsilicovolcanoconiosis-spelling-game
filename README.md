## 專案技術
- 前端：HTML、CSS、jQuery
- 部署：GitHub Pages
- 版本控制：Git

## 使用技術詳細說明
- 以HTML、CSS及JavaScript撰寫，為一頁式靜態網頁
- 應用RWD以使網頁適應各尺寸裝置瀏覽
- 應用Flexbox技術操縱部分元素
- 使用ES6、jQuery使JavaScript語法變得簡潔（箭頭函式、簡化DOM控制等）
- 採用UJS寫法，維持HTML簡潔
- 運用AJAX promise設計遊戲機制
